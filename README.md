Multimodal Emotion Recognition & Empathetic Dialogue System

An AI-powered multimodal emotion recognition system that analyzes voice, text, and facial expressions to detect human emotions and generate real-time empathetic responses using advanced Deep Learning, NLP, and Computer Vision techniques.


🚀 Features

🎤 Emotion detection from voice input

😊 Facial emotion recognition using FER+ and MTCNN

💬 Text-based emotion analysis using DistilRoBERTa

🧠 Multimodal emotion fusion for accurate prediction

🤖 Real-time empathetic AI response generation

🔒 Privacy-focused and lightweight architecture

⚡ Fast inference with Groq Cloud integration

🌐 Interactive Streamlit web interface




🛠️ Technologies Used

Programming Language : Python, Libraries & Frameworks, Streamlit, DeepFace, OpenCV, TensorFlow, Transformers, SpeechRecognition, NumPy, Pandas, AI Models, DistilRoBERTa,  FER+, MTCNN, Groq Whisper, Mistral-7B-Instruct

 
📌 System Workflow

User provides:
Voice input

Facial expression

Text message

Speech-to-text conversion

AI generates:

Emotionally aware response

Human-like conversational output


⚙️ Installation

1️⃣ Clone Repository : git clone https://github.com/NadendlaKulayappa/emotion-recognition-system.git

2️⃣ Open Project Folder : cd emotion-recognition-system

3️⃣ Install Dependencies : pip install -r install.txt

🔑 Environment Variables

Create a .env file:

GROQ_API_KEY=your_api_key_here

▶️ Run Project : python -m streamlit run frontend_app.py
